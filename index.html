AI Foundations Pathway

Welcome to the AI Foundations pathway! This track is designed for educators who are new to AI and want to learn how to create effective prompts and generate useful teaching resources.

Learning Objectives:

By the end of this pathway, you will:

  • Understand the principles of effective prompt writing
  • Create clear, specific prompts that generate useful responses
  • Use AI to generate lesson plans, worksheets, and teaching materials
  • Learn to refine and iterate on AI-generated content

Activities:

Your pathway includes:

  • Introduction to AI prompt writing techniques (30 minutes)
  • Hands-on practice with ChatGPT and Claude (45 minutes)
  • Creating subject-specific teaching resources (60 minutes)
  • Sharing and refining AI-generated materials (30 minutes)

Resource Builder Pathway

Welcome to the Resource Builder pathway! This track is designed for educators who want to create interactive learning materials using pre-made AI tools and platforms.

Learning Objectives:

By the end of this pathway, you will:

  • Create interactive quizzes using AI-powered platforms
  • Design engaging educational games for your classroom
  • Develop multimedia learning resources with AI assistance
  • Customize existing tools to fit your teaching needs

Activities:

Your pathway includes:

  • Introduction to AI-powered educational platforms (30 minutes)
  • Creating interactive quizzes with AI tools (45 minutes)
  • Designing educational games and activities (60 minutes)
  • Collaborative resource development workshop (45 minutes)

Code Creator Pathway

Welcome to the Code Creator pathway! This track is designed for educators who want to leverage AI to create custom HTML applications, interactive quizzes, and games by generating and modifying code.

Learning Objectives:

By the end of this pathway, you will:

  • Use AI to generate functional HTML, CSS, and JavaScript code
  • Create custom interactive quizzes with multimedia elements
  • Develop educational games tailored to your subject area
  • Learn to modify and troubleshoot AI-generated code

Activities:

Your pathway includes:

  • Introduction to AI code generation (30 minutes)
  • Creating custom HTML quizzes with AI (60 minutes)
  • Developing interactive educational games (60 minutes)
  • Troubleshooting and modifying code workshop (45 minutes)
