Teacher's Prompting Cheat Sheet

A collection of ready-to-use prompts to jump-start your AI journey. Simply copy, paste, and customize to fit your specific needs.

How to Use This Cheat Sheet

These prompts are starting points—not perfect solutions. You'll need to add your own context and refine them through iteration. Each prompt is designed to be easily customizable for your specific subject, year group, and teaching goals.

1

Copy

Select a prompt that matches your resource needs and click the "Copy" button.

2

Customize

Replace the placeholder information with your specific subject, year group, and context.

3

Iterate

Refine the AI's response with follow-up prompts until you get exactly what you need.

Resource Creation

Basic Worksheet

Create a worksheet for Year [X] [SUBJECT] on [TOPIC]. The worksheet should include: - A brief introduction to the topic - 10 questions that progress from basic recall to application - A challenge section for advanced students - Key vocabulary definitions - A visual element (described in text form) Format this as a ready-to-print worksheet with appropriate spacing and sections. Use British English spelling and terminology.

Age-appropriate: vocabulary and sentence structure matching Year [X] expected level 3. Extended version: more sophisticated language, additional detail and nuance, challenging vocabulary Each text should be approximately [X] words and cover the same key information points: - [KEY POINT 1] - [KEY POINT 2] - [KEY POINT 3] Include a brief explanation of how each version differs in complexity. Knowledge Organiser

The knowledge organiser should fit on a single A4 page and include: - Key terminology with definitions (minimum 10 terms) - Core knowledge and facts in concise bullet points - Important dates/figures/formulas as relevant - A visual element that clarifies a central concept (described in text form) - Common misconceptions and their corrections - Links to prior and future learning - Sample exam questions or knowledge check questions - Suggestions for additional resources or further reading Format this in a visually organised way that would be easy to follow when printed. Use British English spelling and terminology.

Revision Cards Set

A concise, well-structured answer on the back The cards should: - Cover the key knowledge from the [EXAM BOARD] specification - Progress from foundational understanding to application and analysis - Include a mix of definitions, explanations, applications, and evaluations - Incorporate relevant examples, diagrams, or formulas where appropriate - Address common misconceptions or tricky concepts Format these as a table with two columns: "Card Front" and "Card Back". Use British English spelling and terminology.

Scaffolded Writing Framework

The framework should include: - Clear explanation of the text type's purpose and structure - Success criteria aligned with [CURRICULUM/EXAM BOARD] expectations - Annotated model example showing key features - Vocabulary bank organised by categories/paragraphs - Sentence starters and connectives appropriate for this text type - Planning templates with guided prompts - Paragraph-by-paragraph scaffolding with guided questions - Editing and proofreading checklist Create three differentiated versions: 1. Highly scaffolded for struggling writers 2. Moderately scaffolded for mid-level writers 3. Lightly scaffolded for confident writers Format this as a complete resource pack that a teacher could print and use immediately. Use British English spelling, punctuation and terminology.

Assessment & Feedback

Multiple Choice Quiz

Create a 20-question multiple choice quiz for Year [X] [SUBJECT] on [TOPIC]. For each question: - Provide a clear question stem - Include 4 options (1 correct, 3 plausible distractors) - Mark the correct answer - Provide a brief explanation of why the answer is correct The questions should range from basic recall to application and analysis, covering the key knowledge points: - [KEY POINT 1] - [KEY POINT 2] - [KEY POINT 3] Format this as a clean, professional-looking document using British English spelling and terminology.

Mock Exam Paper

Create a [DURATION]-minute mock exam paper for [SUBJECT] [EXAM LEVEL] following the [EXAM BOARD] style and format. I've attached the specification points to be covered. The paper should include: - A realistic mix of question types matching the official exam structure - Questions covering all assessment objectives (AO1, AO2, AO3, etc.) - The correct mark allocation for each question - A range of difficulty levels progressing through the paper - Clear instructions for students - Any resource material needed (text extracts, data, etc.) Also provide: - A detailed mark scheme with example answers - A grade boundary estimation based on previous years - Common student misconceptions or errors to watch for Format this as an authentic-looking exam paper that follows the conventions of the examination board. Use British English spelling, punctuation and terminology throughout.

Detailed Marking Rubric

Create a detailed marking rubric for a Year [X] [SUBJECT] [ASSIGNMENT TYPE] on [TOPIC]. The rubric should: - Include 4-5 key assessment criteria (e.g., content knowledge, analysis, communication, etc.) - Have 4 performance levels for each criterion (e.g., emerging, developing, secure, mastery) - Contain clear, specific descriptors for each level of each criterion - Include a points/grade allocation system - Be aligned with the [CURRICULUM/EXAM BOARD] expectations for this age group - Be written in student-friendly language so it can be shared with students before the assignment Also provide: - 2-3 example comments for each performance level to help me provide consistent feedback - Suggestions for how to use this rubric as a self-assessment tool with students Format this in a visually clear table format suitable for printing. Use British English spelling and terminology throughout.

A mid-unit peer assessment activity with structured feedback prompts 4. An exit ticket system with graduated questions for 5 different lessons 5. A self-assessment reflection tool aligned to learning objectives For each assessment tool, provide: - Clear teacher instructions for implementation - All necessary student-facing content - Success criteria or answers where relevant - Guidance on how to use the assessment data to inform teaching - Follow-up questions or activities based on possible outcomes Design these to be visually appealing, easy to implement, and provide meaningful data on student understanding. Personalised Feedback Generator

Differentiation & Inclusive Resources

Tiered Task Generator

Create a set of differentiated learning tasks for Year [X] [SUBJECT] on [TOPIC] that allows all students to access the same key learning but with appropriate levels of challenge. Design three clearly differentiated tiers: 1. Foundation tier - For students working below age-related expectations 2. Core tier - For students working at age-related expectations 3. Extension tier - For students working above age-related expectations Each tier should: - Focus on the same key learning objectives aligned with [CURRICULUM/EXAM BOARD] - Provide an appropriate level of support/challenge - Include clear success criteria specific to that tier - Build student confidence while maintaining high expectations For each tier, provide: - A complete set of student instructions and tasks - Any necessary resources or templates - Assessment criteria specific to that tier - Common misconceptions to watch for at this level - Guidance on how to support students at this level Also include a system for: - How students should be assigned to tiers - How to manage the different groups during the lesson - How students can self-select appropriate challenge - How to enable progression between tiers Format this as a complete teaching resource pack ready for classroom use. Use British English spelling and terminology throughout.

Inclusive Resource Adaptation

Adapt this [RESOURCE TYPE] on [TOPIC] to make it fully inclusive and accessible for a diverse class that includes students with the following needs: - Students with dyslexia or reading difficulties - English as an Additional Language (EAL) learners at early acquisition stage - Students with ADHD and focus challenges - Students with working memory difficulties - Students with visual processing challenges - Students with auditory processing difficulties Make adaptations that maintain the learning objectives and content, but modify presentation, instruction, and response methods to ensure all students can access the material. Include: 1. Universal Design for Learning (UDL) principles applied throughout: - Multiple means of representation - Multiple means of action and expression - Multiple means of engagement 2. Specific modifications such as: - Visual supports (described in detail) - Simplified language options - Chunked instructions - Processing time accommodations - Alternative recording methods - Memory aids and reference materials 3. A guide for implementing these adaptations in whole-class teaching 4. Additional support strategies for teaching assistants 5. Digital and non-digital options for adaptations The adaptations should maintain high expectations for all learners while removing barriers to participation. Use British English spelling and terminology throughout.

Progressive Scaffolding Framework

Create a comprehensive scaffolding framework for teaching [TOPIC] to Year [X] [SUBJECT] students with diverse learning needs, following [CURRICULUM/EXAM BOARD] requirements. Design a progressive scaffolding system that: 1. Starts with maximum support for initial learning 2. Gradually removes scaffolding as students develop competence 3. Eventually leads to independent application of skills/knowledge Include the following scaffolding elements, with examples for each stage of learning: 1. Cognitive scaffolds: - Thinking frameworks and graphic organisers - Worked examples with explanations - Partial examples requiring completion - Self-questioning prompts - Analogies and connections to prior knowledge 2. Procedural scaffolds: - Step-by-step guides - Checklists and success criteria - Process maps or flowcharts - Models of excellence with annotations 3. Metacognitive scaffolds: - Self-monitoring tools - Reflection questions and prompts - Error analysis frameworks - Strategy selection guides 4. Conceptual scaffolds: - Key concept explanations at different levels - Visual representations of abstract ideas - Concrete to abstract learning progression - Connections between concepts For each type of scaffold, provide: - The specific learning phase where it's most useful - Implementation guidance for teachers - Criteria for when to remove or reduce the scaffold - Alternative approaches for different learning preferences The framework should be structured to gradually transfer responsibility from teacher to student. Use British English spelling and terminology throughout.

Challenge Extension Pack

Create a comprehensive extension pack for highly able/gifted Year [X] [SUBJECT] students who have mastered the core curriculum content on [TOPIC]. This should extend their learning horizontally (breadth) rather than simply accelerating to the next year's content. The extension pack should include: 1. Depth extension activities that: - Develop higher-order thinking skills (analysis, evaluation, synthesis) - Explore underlying principles and concepts - Examine alternative perspectives or interpretations - Encourage critical evaluation of the content - Promote metacognitive awareness and reflection 2. Complexity extension activities that: - Introduce interdisciplinary connections - Present multi-step problems requiring strategic thinking - Incorporate ambiguity and multiple variables - Challenge students to devise their own solutions - Involve systems thinking and complex relationships 3. Creative application activities that: - Encourage innovative problem-solving - Allow for multiple correct approaches - Provide opportunities for student-led inquiry - Require adaptation of knowledge to novel situations - Result in original student products or ideas Also include: - Assessment criteria that recognise sophisticated understanding - Guidance for teachers on facilitating rather than directing - Self-assessment tools for students - Links to advanced resources and references - Suggestions for authentic audiences for student work The pack should maintain connection to the core curriculum while offering significant additional challenge. Use British English spelling and terminology throughout.

Layout & Design Prompts

Professional Document Formatting

Please enhance the formatting and layout of this [RESOURCE TYPE] to make it look professional and engaging. Use a clean, visually appealing design that would be suitable for a high-quality teaching resource. Include: - Clear, hierarchical headings and subheadings - Appropriate use of white space and margins - Consistent font choices (suggest specific professional fonts) - Thoughtful use of formatting elements like bold, italics, and bullet points - A unified colour scheme (suggest 2-3 complementary colours) - Page numbering and headers/footers if appropriate - Any design elements that would aid comprehension and engagement Format this for [PURPOSE: e.g., student handout, presentation, teacher reference, etc.] using A4 page size. Use British English spelling and conventions throughout. Avoid any formatting that looks obviously AI-generated – aim for a design that appears professionally created with natural variation in structure and presentation.

The current content is good but needs to be reformatted to appeal to [AGE RANGE] students while maintaining educational value. Please implement a design that includes: - An age-appropriate visual theme related to [SUBJECT/TOPIC] - Text formatting that supports readability for this age group (appropriate font size, line spacing, etc.) - Strategic use of colours, shapes, or design elements to organise information - Visual cues that highlight key information - Thoughtful balance between visual appeal and educational content - Layout elements that guide the reader through the content logically - Any additional features that would engage students of this age group The design should look authentic and professionally created, not like an AI-generated template. Incorporate design principles that would be found in high-quality educational resources from established publishers. Use British English spelling and terminology throughout.

Final Tip:

Remember that these prompts are starting points. They'll almost never give you exactly what you need on the first try. Be prepared to iterate, provide feedback, and refine the AI's output. The AI gets smarter with your guidance—think of it as training a new teaching assistant who needs clear direction to produce exactly what you want.

Use these prompts as inspiration for your own AI experiments. The more you practice, the more effective your prompting will become.
